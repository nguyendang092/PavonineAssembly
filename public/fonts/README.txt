Đặt file font TDF_cut-putroe-navisha-kyokm3.otf vào thư mục này.
Sau đó, thêm đoạn sau vào file CSS (ví dụ: src/styles.css hoặc src/index.css):

@font-face {
  font-family: 'TDF_cut-putroe-navisha-kyokm3';
  src: url('/fonts/TDF_cut-putroe-navisha-kyokm3.otf') format('opentype');
  font-weight: normal;
  font-style: normal;
}

Sau đó, bạn có thể dùng font này trong CSS hoặc canvas với tên 'TDF_cut-putroe-navisha-kyokm3'.
